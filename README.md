# streamlit_prac

## Usage

`pip install streamlit`

1. run locally  
`git clone https://github.com/teacher144123/streamlit_prac.git`  
`cd streamlit_prac/`  
`streamlit run YOUR_SCRIPT.py`

2. run remotely  
`streamlit run https://github.com/teacher144123/streamlit_prac/blob/master/min3.py`

## Examples
